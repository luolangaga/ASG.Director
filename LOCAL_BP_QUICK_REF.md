# 本地BP比分板修复 - 快速参考

## 修复内容

✅ **修复了打开比分板时会自动打开后台的问题**
✅ **修复了只能打开B队比分板的bug（team参数错误）**
✅ **新增一键打开双比分板功能**
✅ **修复了打开赛后数据时会自动打开后台的问题**

## 新功能

### 比分管理页面
- 🔵 **打开A队比分板** - 单独打开A队比分板
- 🔴 **打开B队比分板** - 单独打开B队比分板
- 🟢 **一键打开双比分板** - 同时打开A队和B队比分板（新增）

### 赛后数据页面
- 📊 **打开赛后数据窗口** - 直接打开赛后数据展示窗口

### 后台控制
- 如需打开后台窗口，请使用菜单中的"比分控制"或"赛后数据"按钮

## 使用方法

### 场景1：只需要比分板
1. 进入"比分管理"页面
2. 点击"一键打开双比分板"
3. 两个比分板窗口会同时打开，无需打开后台

### 场景2：需要编辑比分
1. 进入"比分管理"页面
2. 在页面中编辑比分数据
3. 点击"保存"
4. 比分板会自动更新

### 场景3：需要赛后数据
1. 进入"赛后数据"页面
2. 填写或使用OCR识别数据
3. 点击"保存"
4. 点击"打开赛后数据窗口"查看效果

## 技术细节

- **修改文件**: `pages/local-bp.html`
- **team参数**: 已统一为 `'teamA'` 和 `'teamB'`
- **API调用**: 使用 `window.electronAPI.openScoreboard(roomId, team)`
- **向后兼容**: 所有修改都保持向后兼容
