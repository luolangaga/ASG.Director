# ASG Director Blender 插件（BP 角色同步）

文件：`asg_director_bp_sync_addon.py`

## 功能

- 绑定五个槽位：求生者1、求生者2、求生者3、求生者4、监管者。
- 从 ASG.Director 的 BP 状态同步角色。
- 按统一目录规则加载模型：
  - `模型根目录/{角色名称}/`
  - 在该目录内扫描 `.gltf` 文件，按文件名排序后使用第一个。
- 导入后自动设置动作循环（对导入动作添加 Cycles）。
- 启动同步后自动弹出一个新窗口，切到“相机视角 + 渲染模式”（开箱即用预览）。
- 自动尝试重连缺失贴图（在角色目录内递归查找同名贴图文件）。
- 自动隐藏常见辅助物体（如名称包含 sphere/collision/helper 等）。

## 安装

1. Blender -> `Edit` -> `Preferences` -> `Add-ons` -> `Install...`
2. 选择 `asg_director_bp_sync_addon.py`
3. 勾选启用插件

## 使用

1. 在 3D 视图侧边栏打开 `ASG` 面板。
2. 设置：
   - `BP API`：默认 `http://127.0.0.1:9528/api/current-state`
   - `模型根目录`：你的自定义目录
3. 点 `绑定默认五槽位`（会创建并绑定 5 个 Collection）。
4. 点 `启动同步`，插件会按轮询间隔自动同步 BP 选角并替换对应模型。
5. 也可以点 `立即同步` 手动触发一次。

## 说明

- 依赖 ASG.Director 本地页面服务的 `api/current-state` 接口。
- 若模型未找到，会在面板状态中显示错误路径信息。
- 若你不想自动弹出渲染窗口，可在面板里关闭“启动同步时自动打开渲染窗口”。
